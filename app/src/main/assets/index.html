Властивості кореня n-го степеня

Коренем n-го степеня з числа а називається таке число, n-й степінь якого дорівнює а, тобто:

Теорія

Знаходження кореня n-го степеня з числа а називають добуванням кореня n-го степеня, число а називають відкореневим числом, а число n - показником кореня.

У випадку якщо a < 0 , то корінь n-го степеня цього числа не існує.

* при n = 3, замість «корінь третього степеня» можна сказати «корінь кубічний».

Наприклад

Корінь четвертого степеня від числа 81 дорівнює 3  

, так як 34 = 81.

Числові послідовності (Арифметична прогресія)

Послідовність, де кожен член починаючи з другого можна знайти додавши до попереднього одне й те саме число d, називають арифметичною прогресією.

Теорія

Число d називають різницею арифметичної прогресії. n - номер члена, аn - n-й член.

Наприклад

1) Арифметична прогресія 1, 2, 3, 4,...n,... , у якій а1 = 1, d = 1.
2) Арифметична прогресія 2, 4, 6, 8,...n,... , у якій а1 = 2, d = 2.

Формули з поясненнями

Формула знаходження наступного члена прогресії після n-ого члена прогресії.

Формула знаходження n-ого члена прогресії за допомогою різниці прогресії і першого члена.

Формула знаходження n-ого члена прогресії за допомогою середнього арифметичного двох сусідніх членів.

Формула знаходження суми перших n членів арифметичної прогресії за допомогою середнього арифметичного першого і n-го членів прогресії помножену на n.

Формула знаходження суми перших n членів арифметичної прогресії іншим способом.
